Visma Identity App

Description

This is a demonstration of the Visma Identity App, which allows users to confirm payments, sign documents, and login to various services using a custom URI scheme. The app is built using JavaScript and allows users to test different URIs by entering them in the input field or by clicking on the provided buttons. The URI is parsed and validated using regular expressions, and the path and parameters are displayed on the page. This is only a demonstration and should not be used in production. This app is open-source and available for anyone to use, modify and distribute.
